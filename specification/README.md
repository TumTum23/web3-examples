# Web3 Specification

`` Version: 0.0.1 ``

The Web3 specification will document the following points:

- Library Architecture
- Security Concept
- Quality Assurance
- Public API Concept

## Contributing

### Prerequisites

NodeJS, Python, git (for git bash), and npm are required to run and test.

### Getting Started

Install docsify and serve the docs locally.

```bash
npm install
npm run compile
npm run start
```
